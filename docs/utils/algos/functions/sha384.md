# Function: sha384()

> **sha384**(`input`): `Promise`\<[`ByteArray`](../../../types/type-aliases/ByteArray.md)\>

Computes the SHA-384 hash of the input data.

## Parameters

### input

[`ByteArray`](../../../types/type-aliases/ByteArray.md)

The data to hash.

## Returns

`Promise`\<[`ByteArray`](../../../types/type-aliases/ByteArray.md)\>

A promise that resolves to the SHA-384 hash as a byte array.

## Example

```typescript
const hash = await sha384(new Uint8Array([1, 2, 3]))
```
