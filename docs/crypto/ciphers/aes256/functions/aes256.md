# Function: aes256()

> **aes256**(`key`): [`Cipher`](../../../types/interfaces/Cipher.md)

Creates an AES-256-CBC cipher for PDF encryption.
The cipher prepends a zero IV to encrypted data during encryption
and extracts the IV from the first 16 bytes during decryption.

## Parameters

### key

[`ByteArray`](../../../../types/type-aliases/ByteArray.md)

The 32-byte encryption key.

## Returns

[`Cipher`](../../../types/interfaces/Cipher.md)

A Cipher object with encrypt and decrypt methods.

## Example

```typescript
const cipher = aes256(key)
const encrypted = await cipher.encrypt(plaintext)
const decrypted = await cipher.decrypt(encrypted)
```
